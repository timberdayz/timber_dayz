# CLAUDE.md

This repository uses `AGENTS.md` as the single active rule entrypoint.

Claude is a supplemental assistant for review, consultation, and second opinions. When using Claude for this repository:

- Follow `AGENTS.md` first.
- Do not treat `.cursorrules` or `.cursor/rules/*` as active rule sources.
- Do not duplicate repository workflow, architecture, or release rules in this file.
- Use `docs/DEVELOPMENT_RULES/README.md` for detailed implementation references.
- Use `docs/guides/DEVELOPMENT_WORKFLOW.md` for commands.
- Use `docs/guides/PRE_LAUNCH_RULES.md` for launch-period constraints.
- Use `docs/architecture/README.md` for architecture context.

Always answer users in Chinese unless they explicitly ask for another language.
