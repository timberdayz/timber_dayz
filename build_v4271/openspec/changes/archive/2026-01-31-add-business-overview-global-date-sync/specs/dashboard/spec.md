# Spec Delta: 数据看板能力

## ADDED Requirements

### Requirement: 业务概览全局日期与模块同步

系统 SHALL 在业务概览页面提供页面级全局日期组件，支持粒度（日/周/月）与日期选择；默认各模块（数据对比、店铺赛马、流量排名、经营指标）跟随全局日期；用户可手动修改某模块日期以覆盖全局，并可恢复「跟随全局」。

#### Scenario: 页面加载默认

- **WHEN** 用户打开业务概览页面
- **THEN** 全局日期组件显示粒度=月、日期=当前月份
- **AND** 数据对比、店铺赛马、流量排名、经营指标模块的日期与全局一致
- **AND** 各模块数据按此日期加载

#### Scenario: 全局日期变更同步

- **WHEN** 用户修改全局日期组件的粒度或日期
- **THEN** 所有「跟随全局」的模块的日期/粒度同步更新
- **AND** 各模块重新加载数据
- **AND** 已手动覆盖的模块不受影响

#### Scenario: 单模块手动覆盖

- **WHEN** 用户直接修改某模块（如数据对比）的日期或粒度
- **THEN** 该模块设置为「不跟随全局」
- **AND** 该模块按用户选择加载数据
- **AND** 后续全局日期变更不影响该模块

#### Scenario: 恢复跟随全局

- **WHEN** 用户在某模块点击「跟随全局」或链接图标
- **THEN** 该模块恢复为跟随全局状态
- **AND** 该模块日期/粒度与全局一致
- **AND** 该模块重新加载数据

#### Scenario: 经营指标日期映射

- **WHEN** 全局粒度为周且经营指标跟随全局
- **THEN** 经营指标使用的日期为该周周一（YYYY-MM-DD）
- **WHEN** 全局粒度为月且经营指标跟随全局
- **THEN** 经营指标使用的日期为该月1日（YYYY-MM-01）

#### Scenario: 时间约定一致性

- **WHEN** 全局或各模块选择周粒度
- **THEN** 周范围为一周周一至周日
- **WHEN** 全局或各模块选择月粒度
- **THEN** 月范围为当月1日至月末
