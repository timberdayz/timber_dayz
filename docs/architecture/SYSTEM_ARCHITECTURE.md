# 🏢 现代化跨境电商ERP系统设计

## 🎯 系统架构设计

### 核心设计理念
- **数据驱动决策**: 以数据看板为核心，支持实时监控和智能分析
- **角色权限管理**: 基于角色的权限控制，支持管理员、主管、操作员等角色
- **模块化设计**: 高度模块化，支持灵活扩展和定制
- **现代化UI/UX**: 采用最新的设计趋势和用户体验标准

### 1.5 🔍 智能字段映射审核 (Field Mapping Review) ⭐ v4.2.0新增
**目标**: 实现跨平台数据标准化，自动化数据入库流程

**功能特性**:
- 🎯 **智能文件扫描** - 自动发现和注册待处理数据文件
- 📊 **多粒度支持** - 日度/周度/月度数据自动识别和过滤
- 👁️ **实时数据预览** - 预览文件前100行数据
- 🤖 **AI字段映射** - 智能识别和建议字段映射关系
- ✅ **人机协同审核** - 支持手动调整和确认映射规则
- 💾 **模板管理** - 保存和复用字段映射模板
- 🚀 **批量处理** - 支持多文件批量映射和入库
- 📈 **Catalog状态监控** - 实时显示文件处理状态统计

**技术架构**:
- **前端**: Vue.js 3 + Element Plus + Pinia
- **路由**: Hash模式（支持`#/field-mapping`深链接）
- **后端**: FastAPI + vue_field_mapping应用
- **数据库**: SQLite catalog_files表
- **AI引擎**: 智能字段匹配算法

**访问方式**:
- 深链接: `http://localhost:5173/#/field-mapping`
- 侧边栏菜单: "字段映射审核"（开发模式自动置顶）

## 🎨 角色权限设计

### 管理员 (Administrator)
- **权限范围**: 全部功能模块
- **特殊权限**: 系统设置、用户管理、数据管理
- **数据访问**: 所有数据，包括敏感财务数据

### 主管 (Manager)
- **权限范围**: 业务概览、销售分析、库存管理、人力管理、店铺管理
- **限制权限**: 无系统设置权限，无用户管理权限
- **数据访问**: 本部门数据 + 汇总数据

### 操作员 (Operator)
- **权限范围**: 基础功能模块
- **限制权限**: 只读权限为主，部分编辑权限
- **数据访问**: 本人相关数据

### 财务专员 (Finance Specialist)
- **权限范围**: 财务管理、部分销售分析
- **特殊权限**: 财务报表、成本分析
- **数据访问**: 财务相关数据

### 库存专员 (Inventory Specialist)
- **权限范围**: 库存管理、部分销售分析
- **特殊权限**: 采购管理、库存优化
- **数据访问**: 库存相关数据
