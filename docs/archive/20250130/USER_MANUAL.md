## 📊 核心功能使用

### 1. 数据看板

**查看销售数据**:
1. 点击左侧菜单"销售看板"
2. 选择日期范围和平台
3. 查看实时数据展示
4. 点击"刷新数据"获取最新数据

**筛选功能**:
- 日期范围: 选择查询的日期区间
- 平台: 选择Shopee、TikTok、Amazon等
- 粒度: 选择日度、周度、月度数据

### 2. 数据采集

**数据采集流程**:
1. 进入"数据采集中心"
2. 选择要采集的平台
3. 选择要采集的账号和店铺
4. 设置采集参数
5. 点击"开始采集"
6. 等待采集完成

**注意事项**:
- 采集时间取决于数据量
- 不要同时进行多个采集任务
- 采集完成后会自动保存数据

### 3. 数据管理

**上传Excel文件**:
1. 进入"字段映射审核"
2. 点击"上传文件"
3. 选择Excel文件
4. 等待文件上传完成
5. 系统会自动识别字段

**字段映射**:
1. 查看系统识别的字段映射
2. 检查映射是否正确
3. 手动调整不正确的映射
4. 点击"确认映射"
5. 开始数据导入

**数据验证**:
- 系统会自动验证数据格式
- 如有错误会显示错误信息
- 修正错误后重新导入

### 4. 库存管理

**查看库存**:
1. 进入"库存管理"
2. 查看当前库存列表
3. 可以按SKU、仓库筛选
4. 查看库存详情

**库存调整**:
1. 选择要调整的商品
2. 点击"调整库存"
3. 输入调整数量和原因
4. 确认调整

**库存预警**:
- 系统会自动检测库存不足
- 红色标记表示库存告警
- 及时补充库存

### 5. 财务管理

**应收账款**:
1. 进入"财务管理"
2. 查看应收账款列表
3. 可以按状态筛选
4. 记录收款信息

**财务报表**:
- 利润分析报表
- 收支明细报表
- 资金流水报表
- 导出报表功能

### 6. 用户管理

**创建用户**（管理员权限）:
1. 进入"用户管理"
2. 点击"新增用户"
3. 填写用户信息
4. 分配角色和权限
5. 保存用户

**修改用户**:
1. 在用户列表中找到用户
2. 点击"编辑"
3. 修改用户信息
4. 保存修改

**重置密码**:
1. 点击用户的"重置密码"
2. 输入新密码
3. 确认重置

## ❓ 常见问题

### 1. 登录问题

**Q: 忘记密码怎么办？**  
A: 联系管理员重置密码。

**Q: 无法登录系统？**  
A: 检查网络连接，确认用户名密码正确，如仍无法登录请联系管理员。

### 2. 数据问题

**Q: 数据没有更新？**  
A: 点击"刷新数据"按钮，如果还是没有更新，检查数据采集是否完成。

**Q: 导入数据失败？**  
A: 检查Excel格式是否正确，确认字段映射无误，查看错误提示信息。

### 3. 权限问题

**Q: 无法访问某个功能？**  
A: 可能是权限不足，请联系管理员分配相应权限。

**Q: 无法导出数据？**  
A: 确认是否有导出权限，检查数据是否为空。

### 4. 性能问题

**Q: 系统运行缓慢？**  
A: 清除浏览器缓存，关闭不必要的标签页，如果持续缓慢请联系技术支持。

**Q: 文件上传失败？**  
A: 检查文件大小（限制100MB），确认网络连接稳定，尝试重新上传。
