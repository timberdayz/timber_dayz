# 🚀 店铺管理系统快速演示指南

## 📖 系统概述

本系统专为跨境电商卖家设计，支持多平台多店铺的统一管理。通过专业的Web界面，您可以轻松管理Shopee、Amazon、Lazada等平台的多个店铺账号。

### 🎯 核心特色

- **🏪 店铺级数据隔离**: 每个店铺的数据完全独立存储
- **📊 健康度监控**: 实时监控店铺运营健康状况
- **☁️ 云端优化**: 针对2核3M带宽云端环境专门优化
- **🔄 批量管理**: 支持多账号批量操作和状态管理

## 🚀 5分钟快速体验

### 步骤1：环境准备（2分钟）

```bash
# 1. 安装依赖包
pip install streamlit pandas plotly pyyaml playwright

# 2. 安装浏览器驱动
playwright install

# 3. 验证系统配置
python test_store_management.py
```

### 步骤2：启动系统（30秒）

```bash
# 方式一：统一管理界面（推荐）
streamlit run frontend_streamlit/unified_dashboard.py

# 方式二：独立店铺管理界面
python start_store_management.py
```

### 步骤3：访问界面（30秒）

打开浏览器访问：`http://localhost:8501`

- **统一界面**: 选择"运营中心" → "店铺管理"
- **独立界面**: 直接进入店铺管理中心

### 步骤4：添加店铺（2分钟）

1. **选择平台**: 点击"Shopee"标签页
2. **添加账号**: 点击"➕ 添加新店铺账号"
3. **填写信息**:
   ```
   店铺名称: MyStore_SG
   用户名: your_shopee_email@example.com
   密码: your_password
   登录URL: https://seller.shopee.sg
   地区: SG
   货币: SGD
   ```
4. **保存账号**: 点击"添加店铺账号"

## 💡 使用技巧

### 1. 店铺命名规范
建议使用以下命名格式：
- `MyStore_SG` - Shopee新加坡店铺
- `MyStore_MY` - Shopee马来西亚店铺
- `Amazon_US` - Amazon美国站点
- `Lazada_TH` - Lazada泰国站点

### 2. 健康度评分理解
- **90-100分**: 💚 优秀 - 运行状况良好
- **70-89分**: 💙 良好 - 运行正常
- **50-69分**: 💛 警告 - 需要关注
- **0-49分**: ❤️ 危险 - 需要立即处理

### 3. 云端环境优化建议
- 启用无头浏览器模式节省内存
- 限制最大并发采集器为1个
- 启用数据压缩减少存储占用
- 定期清理临时文件释放空间

### 4. 数据安全注意事项
- 定期备份店铺配置数据
- 不要在生产环境上传账号配置文件
- 使用强密码保护店铺账号
- 定期更换登录密码

## 🔧 故障排除

### 常见问题解决

#### 1. 模块导入失败
```bash
# 解决方案：重新安装依赖
pip install -r requirements.txt --force-reinstall
```

#### 2. 浏览器驱动问题
```bash
# 解决方案：重新安装Playwright
playwright install --force
```

#### 3. 端口占用问题
```bash
# 解决方案：使用不同端口
streamlit run frontend_streamlit/unified_dashboard.py --server.port 8502
```

#### 4. 配置文件错误
```bash
# 解决方案：重新生成配置
python test_store_management.py
```
