# 🌐 现代化ERP前端系统使用指南

## 🎯 系统概述

西虹ERP系统已完成现代化前端重构，采用Vue.js 3 + Element Plus技术栈，实现了专业级的跨境电商ERP管理系统。

### 核心特性
- ✅ **现代化UI设计** - 采用最新设计趋势，专业美观
- ✅ **角色权限管理** - 基于角色的访问控制（RBAC）
- ✅ **数据可视化** - 专业图表展示，实时数据更新
- ✅ **响应式设计** - 支持桌面、平板、移动端
- ✅ **模块化架构** - 高度模块化，易于扩展

## 🚀 快速开始

### 启动前端服务

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖（首次运行）
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问系统
# 浏览器打开: http://localhost:5173
```

### 通过主入口启动

```bash
# 1. 返回项目根目录
cd ..

# 2. 启动主入口
python run_new.py

# 3. 选择 "5" - 前端页面管理
# 4. 选择 "1" - 启动前端服务
# 5. 选择 "5" - 在浏览器中打开
```

## 🔐 权限管理

### 角色定义

#### 管理员 (Administrator)
- **权限范围**: 全部功能模块
- **特殊权限**: 系统设置、用户管理、数据管理
- **数据访问**: 所有数据，包括敏感财务数据

#### 主管 (Manager)
- **权限范围**: 业务概览、销售分析、库存管理、人力管理、店铺管理
- **限制权限**: 无系统设置权限，无用户管理权限
- **数据访问**: 本部门数据 + 汇总数据

#### 操作员 (Operator)
- **权限范围**: 基础功能模块
- **限制权限**: 只读权限为主，部分编辑权限
- **数据访问**: 本人相关数据

#### 财务专员 (Finance Specialist)
- **权限范围**: 财务管理、部分销售分析
- **特殊权限**: 财务报表、成本分析
- **数据访问**: 财务相关数据

#### 库存专员 (Inventory Specialist)
- **权限范围**: 库存管理、部分销售分析
- **特殊权限**: 采购管理、库存优化
- **数据访问**: 库存相关数据

### 权限控制机制
- **路由权限** - 基于角色的路由访问控制
- **组件权限** - 页面内组件级权限控制
- **数据权限** - 数据访问级别权限控制
- **操作权限** - 按钮级操作权限控制

## 📱 响应式设计

### 断点设置
- **桌面端**: ≥1200px - 完整功能展示
- **平板端**: 768px-1199px - 适配布局
- **移动端**: <768px - 简化布局

### 适配策略
- **侧边栏**: 移动端自动隐藏，通过菜单按钮控制
- **图表**: 自动调整尺寸，保持可读性
- **表格**: 支持横向滚动
- **按钮**: 触摸友好的尺寸设计

## 🔧 开发指南

### 添加新页面
1. 在`src/views/`创建新页面组件
2. 在`src/router/index.js`添加路由配置
3. 在侧边栏组件中添加菜单项
4. 配置权限和角色要求

### 添加新图表
1. 安装ECharts依赖
2. 创建图表组件
3. 在页面中引入和使用
4. 配置响应式调整

### 权限控制
1. 在路由meta中配置权限
2. 在组件中使用权限检查
3. 在侧边栏中过滤菜单项

## 🐛 常见问题

### Q1: 页面无法访问
**解决方案**:
- 检查路由配置是否正确
- 确认用户角色权限
- 查看浏览器控制台错误

### Q2: 图表不显示
**解决方案**:
- 确认ECharts已正确安装
- 检查容器尺寸设置
- 查看图表配置是否正确

### Q3: 样式显示异常
**解决方案**:
- 确认Element Plus已正确引入
- 检查CSS变量定义
- 清除浏览器缓存

### Q4: 权限控制不生效
**解决方案**:
- 检查用户角色配置
- 确认路由守卫设置
- 验证权限检查逻辑
