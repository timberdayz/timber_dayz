# ERP 操作培训第一期飞书搭建执行清单

## 目的

本清单用于指导培训负责人或运营同事在飞书中快速搭建 `ERP 操作培训第一期` 的课程与考试结构。

目标是先让培训能真正开始，不等待完整 ERP 培训模块开发完成。

## 一、学习项目结构建议

建议在飞书中建立一个完整学习项目：

- 学习项目名称：`ERP 操作培训第一期`
- 学习对象：新员工 / 首次接触 ERP 的员工
- 完成标准：完成全部课程并通过结业考试

## 二、课程单元搭建清单

建议按固定顺序创建以下 6 个课程单元。

### 课程单元 01：系统登录与基础导航

建议内容：
- ERP 是做什么的
- 如何登录
- 首页结构与导航
- 个人信息与基础设置

准备项：
- [ ] PPT / 图文讲义
- [ ] 登录页截图
- [ ] 首页截图
- [ ] 导航标注图

### 课程单元 02：任务中心与通知中心

建议内容：
- 什么是任务
- 什么是通知
- 如何查看待办
- 如何从通知进入任务或业务页

准备项：
- [ ] PPT / 图文讲义
- [ ] 任务页截图
- [ ] 通知中心截图
- [ ] 标准处理路径示意图

### 课程单元 03：基础数据查询

建议内容：
- 查询的标准路径
- 常见筛选方式
- 列表与详情查看逻辑

准备项：
- [ ] 图文讲义
- [ ] 查询页面截图
- [ ] 筛选区标注图

### 课程单元 04：常用业务页面操作

建议内容：
- 页面结构认知
- 标准操作顺序
- 保存 / 提交 / 检查结果

准备项：
- [ ] 图文讲义
- [ ] 典型业务页面截图
- [ ] 操作流程图

### 课程单元 05：操作规范与留痕意识

建议内容：
- 为什么要留痕
- 哪些动作必须规范
- 备注怎么写
- 常见错误示例

准备项：
- [ ] 图文讲义
- [ ] 错误案例图
- [ ] 备注示例图

### 课程单元 06：异常与升级处理

建议内容：
- 什么是异常
- 发现异常后的标准动作
- 什么情况必须升级
- 升级入口和责任人

准备项：
- [ ] 图文讲义
- [ ] 异常处理流程图
- [ ] 升级路径说明

## 三、课程内容形式建议

第一轮试点不需要重制作，只需要保证内容能被看懂。

推荐最低配置：
- 每个单元 1 份 PPT 或图文讲义
- 关键单元增加短录屏
- 内容以真实系统截图为主

推荐时长：
- 单个单元 5 到 15 分钟
- 整体学习时间控制在 45 到 90 分钟之间

## 四、飞书考试搭建清单

建议创建 1 个结业考试：

- 考试名称：`ERP 操作培训第一期考试`
- 题量：24 题
- 总分：100 分
- 合格线：80 分
- 考试时长：20 到 30 分钟
- 考试次数：2 次

### 推荐题型结构

- 单选题：14 题
- 判断题：6 题
- 场景题：4 题

### 推荐考点分布

- 系统登录与基础导航：15%
- 任务中心与通知中心：25%
- 基础数据查询：20%
- 常用业务页面操作：20%
- 操作规范与留痕意识：10%
- 异常与升级处理：10%

## 五、学习顺序设置建议

建议开启固定学习路径，不建议学员自由乱跳。

顺序如下：
1. 系统登录与基础导航
2. 任务中心与通知中心
3. 基础数据查询
4. 常用业务页面操作
5. 操作规范与留痕意识
6. 异常与升级处理
7. 结业考试

## 七、上线前预演清单

建议用 1 个测试账号完整走一遍。

- [ ] 可以进入课程
- [ ] 可以依次完成课程学习
- [ ] 可以进入考试
- [ ] 可以提交考试
- [ ] 可以查看成绩
- [ ] 负责人可以看到完成结果

## 八、第一轮试点不要追求的内容

第一轮先不要做这些复杂项：
- 积分体系
- 勋章体系
- 复杂证书系统
- 讨论社区
- 多层级审批式学习流程

第一轮只要把下面三件事做好：
- 能学
- 能考
- 能统计结果

## 九、飞书搭建完成标准

当下面条件全部满足时，即可认为飞书侧准备完成：

- [ ] 六个课程单元已建立
- [ ] 所有资料已上传
- [ ] 结业考试已配置
- [ ] 学员访问权限已配置
- [ ] 测试账号已走通全流程

达到以上标准后，即可进入试点发放阶段。
