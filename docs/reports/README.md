# Reports

本目录用于保存历史报告、诊断摘要和阶段性验证结果。

说明：

- 报告会保留运行当时的真实路径、命令和环境信息
- 因此其中出现的 `temp/outputs/` 往往是历史事实，不代表当前正式目录语义
- 当前正式目录语义为：
  - 正式采集原始数据目录：`data/raw/`
  - 工作下载目录：`downloads/`

如果需要判断当前系统是否仍存在双链路问题，请不要只根据报告文本判断，应结合：

- `docs/guides/COLLECTION_PATH_LEGACY_INVENTORY.md`
- 当前活跃 guide / deployment / path 文档
