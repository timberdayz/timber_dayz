# Skill-First Workflow Files

This directory exists to support the active `superpowers` workflow.

## Intended Contents

- `specs/` for brainstorming design documents
- `plans/` for writing-plans output

Create those subdirectories on demand.

## Repository Policy

- `superpowers` and `planning-with-files` are the active workflow tools in this repository
- `openspec/` is preserved separately as historical archive only
- For new features and behavior changes, prefer `brainstorming` -> `writing-plans`
- Use `planning-with-files` for complex refactors, migrations, investigations, and long-running single-person execution
