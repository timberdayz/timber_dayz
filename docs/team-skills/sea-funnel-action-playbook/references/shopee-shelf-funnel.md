# Shopee Shelf Funnel

用于 Shopee 搜索、货架、商品页场景。

## Funnel

```text
曝光 -> 点击 -> 商品页浏览 -> 加购 -> 下单 -> 支付 -> 评价 / 复购
```

## Matrix

| 数据表现 | 瓶颈判断 | 标准动作 | 复查指标 |
|---|---|---|---|
| 搜索曝光高、点击低 | 搜索结果页吸引力不足 | 改首图、标题关键词、价格锚点、优惠标签 | CTR、商品页访问 |
| 点击高、商品页停留弱 | 第一屏卖点不清 | 优化首屏图、核心卖点、价格解释 | 商品页停留、加购率 |
| 商品页浏览高、加购低 | 信任和利益点不足 | 补评价、买家秀、FAQ、保障、视频 | 加购率、咨询率 |
| 加购高、下单低 | 价格、运费、优惠不足 | 调券、包邮、组合、满减 | 下单率、支付率 |
| 下单高、支付低 | 库存、支付、价格临门问题 | 检查库存、运费、优惠可用性、结账路径 | 支付率、取消率 |
| 订单多、评价弱 | 信任资产沉淀不足 | 售后催评、包装卡、评价激励合规设计 | 评价数、评分 |
| 搜索转化好、内容弱 | 货架型机会 | 强化关键词广告、SEO、评价和库存 | 搜索排名、自然流量 |

## Not Recommended

- 曝光高点击低时，不要先补库存。
- 点击高加购低时，不要先扩大广告。
- 加购高支付低时，不要先重拍主图。
