# Signal Action Matrix

本文件是通用信号矩阵。  
若涉及平台细节，再读取 Shopee 或 TikTok 专属矩阵。

| 数据表现 | 说明 | 标准动作 |
|---|---|---|
| 曝光高、点击低 | 主图 / 标题 / 价格 / 卖点不吸引 | 改主图、标题、价格锚点、第一屏卖点 |
| 点击高、加购低 | 商品页信任和利益点弱 | 补详情、评价、视频、FAQ、保障 |
| 加购高、支付低 | 价格、运费、优惠、库存、结账摩擦问题 | 调券、包邮、组合、库存、结账路径 |
| 转化好、毛利差 | 不是精品，是流量品 | 改供应链、提高客单、做套装 |
| 单量多、退货高 | 产品体验或预期管理有问题 | 先修产品、详情和售前说明，不要放大 |
| 内容互动高、成交弱 | 内容吸睛但购买理由弱 | 换脚本，从娱乐转问题解决 |
| 搜索转化好、内容弱 | 更适合 Shopee 搜索和货架 | 加强 SEO、关键词广告、评价沉淀 |
| 内容成交好、搜索弱 | 更适合 TikTok 爆发 | 做达人、短视频、直播和复投 |
| GMV 涨、利润不涨 | 订单质量或成本结构有问题 | 拆价格、优惠、佣金、物流、采购成本 |
| 流量涨、订单不涨 | 流量质量或转化承接弱 | 拆 CTR、PDP 转化、加购、支付 |
| 客单低、订单多 | 低价流量多但精品价值弱 | 做组合、套装、升级款、满减 |
| 库存高、销量弱 | 需求弱或商品承接弱 | 降低补货、做清理、重做卖点或淘汰 |

## Required Output

每条诊断必须输出：

- 当前匹配哪一行信号
- 为什么匹配
- 第一优先动作
- 复查指标
