# TikTok Content Funnel

用于 TikTok 短视频、达人、直播和内容成交场景。

## Funnel

```text
播放 -> 3秒观看 -> 完播 / 互动 -> 商品点击 -> 加购 -> 下单 -> 支付 -> 复投
```

## Matrix

| 数据表现 | 瓶颈判断 | 标准动作 | 复查指标 |
|---|---|---|---|
| 播放高、3秒观看低 | 开头弱，钩子不够 | 重写前 3 秒钩子，直接呈现痛点或结果 | 3 秒观看率 |
| 3秒观看高、完播低 | 内容结构松散 | 缩短脚本、提前展示结果、减少铺垫 | 完播率 |
| 完播高、互动低 | 情绪或参与感不足 | 加提问、对比、评论引导、争议点 | 互动率 |
| 互动高、商品点击低 | 内容吸睛但购买理由弱 | 从娱乐脚本转问题解决脚本，加明确 CTA | 商品点击率 |
| 商品点击高、加购低 | 商品页承接弱 | 优化商品页首图、价格、评价、详情、保障 | 加购率 |
| 加购高、下单低 | 价格和优惠临门问题 | 调券、限时优惠、组合、包邮 | 下单率 |
| 内容成交好、搜索弱 | 内容爆发型机会 | 做达人矩阵、短视频复投、直播承接 | 视频 GMV、直播 GMV |
| 直播进房高、成交弱 | 直播话术和商品讲解弱 | 调整讲品顺序、利益点、限时机制、FAQ | 商品点击、成交率 |
| 达人曝光高、成交弱 | 达人匹配或脚本弱 | 换达人分层、重写脚本、改素材 brief | 达人成交率 |

## Not Recommended

- 播放高但点击低时，不要只加预算。
- 互动高成交弱时，不要继续做纯娱乐内容。
- 商品点击高加购低时，不要先怪达人，先查商品页承接。
