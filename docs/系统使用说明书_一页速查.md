# 西虹ERP · 指标与经营/运营需求速查（一页）

**更新**: 2026-02-02 · 详细说明见 `docs/系统使用说明书.md`

## 指标 → 经营/运营需求 → 入口

| 需求类型 | 你想看的 | 系统指标 | 入口 |
|----------|-----------|----------|------|
| **经营** | 整体业绩达不达标 | 月目标、达成率、时间GAP、今日销售、经营结果 | 工作台 → 业务概览（经营指标） |
| **经营** | GMV/订单/利润健康不 | GMV、订单数、客单价、连带率、人效 | 工作台 → 业务概览（核心KPI） |
| **经营** | 和上期/平均比怎样 | 本期/上期/平均、环比、目标达成率 | 工作台 → 业务概览（数据对比） |
| **经营** | 各店铺/平台谁做得好 | 目标、完成额、完成率、排名 | 工作台 → 业务概览（店铺赛马） |
| **经营** | 设定和跟踪目标 | 销售目标、分解、达成 | 销售与分析 → 目标管理 |
| **经营** | 销售战役效果 | 战役目标、参与店铺、进度 | 销售与分析 → 销售战役管理 |
| **运营** | 流量够不够、谁高 | 访客数、浏览量、排名 | 工作台 → 业务概览（流量排名） |
| **运营** | 转化好不好 | 转化率 | 工作台 → 业务概览（核心KPI） |
| **运营** | 库存积压情况 | 积压商品、积压天数 | 工作台 → 业务概览（库存积压） |
| **运营** | 清仓/滞销优先级 | 清仓排名 | 工作台 → 业务概览（清仓排名） |
| **人力** | 绩效公不公平 | 绩效得分、排名、系数 | 人力资源 → 绩效公示 |
| **人力** | 个人收入明细 | 底薪、绩效、提成、实发 | 人力资源 → 我的收入 |
| **人力** | 谁管哪店、提成比例 | 店铺归属、提成比 | 人力资源 → 人员店铺归属和提成比 |

---

## 按角色怎么用

- **经营/老板**：工作台 → 业务概览（KPI + 数据对比 + 经营指标 + 店铺赛马）；要设目标 → 销售与分析 → 目标管理。
- **运营/店长**：业务概览（流量/库存/清仓/店铺赛马）；销售看板、目标管理、库存管理、店铺分析。
- **数据/采集**：数据采集与管理 → 数据同步（文件列表/同步任务/模板管理）、数据隔离区、数据一致性验证。
- **HR/绩效**：人力资源 → 绩效公示、我的收入、人员店铺归属和提成比、绩效管理（管理员）。

---

## 数据从哪来（一句话）

**配置(A类)** → **Excel 入库/数据同步(B类)** → **系统算指标(C类)** → **业务概览/看板展示**。  
目标在「目标管理」设；订单/产品/流量靠「数据同步」入库；指标才会在业务概览里更新。
