# 飞书培训上传文件

本目录提供运营岗培训一期可直接整理到飞书的本地文件。

文件说明：

- `运营岗培训考试问卷.md`
  用作飞书问卷/表单的题目底稿。建议在飞书中按题目逐项创建。

- `培训完成登记表.csv`
  用作培训完成记录表，可导入飞书多维表格或在线表格。

- `培训反馈表.csv`
  用作培训反馈收集表，可导入飞书多维表格或在线表格。

建议使用方式：

1. 在飞书中创建考试问卷或表单，按 `运营岗培训考试问卷.md` 录入题目。
2. 创建登记表或多维表格时，导入 `培训完成登记表.csv`。
3. 创建反馈表或多维表格时，导入 `培训反馈表.csv`。
4. 上传完成后，把最终 URL 发回给我，我再帮你整理成 ERP 培训配置字段。
