# Sea Business Models

## Status

This is the standard model-selection skill for SEA cross-border e-commerce strategy.  
It is used to compare boutique, curated assortment, and broad listing models with a shared language.

## Core Rule

不要只问“哪种模式赚钱”。  
先判断：

1. 利润来源是什么
2. 团队能力匹配什么
3. 供应链和内容能力能支撑什么
4. 当前阶段应该选哪种模式

## Mandatory Gates

在输出模式判断前，必须完成：

1. 明确当前团队阶段
2. 明确平台与国家
3. 明确当前资源约束：资金、供应链、内容、广告、组织能力
4. 明确目标：短期现金流、利润率、品牌化、规模化中的哪一个优先
5. 输出推荐模式与不推荐模式

## How To Invoke

当用户要求“精品 / 精铺 / 铺货有什么区别”“我们适合哪种模式”“如何从铺货转精品”“哪个模式适合 Shopee / TikTok”时，必须使用本 skill。

涉及选品时补 `sea-product-selection`。  
涉及经营指标和利润结构时补 `sea-metrics-language` 与 `sea-business-review`。

## Standard User Request

```text
请按 team-skills-bundle/sea-business-models/SKILL.md 分析我们当前应该采用哪种跨境电商经营模式。

要求：
1. 对比精品、精铺、铺货
2. 分析利润来源、工作方式、优缺点、适合团队
3. 判断当前阶段更适合哪种模式
4. 如果要转型，给出转型路径
5. 使用 decision-template.md 输出
```

## Output Rule

输出必须包含：

- 当前团队阶段
- 三种模式对比
- 当前推荐模式
- 不推荐原因
- 转型路径
