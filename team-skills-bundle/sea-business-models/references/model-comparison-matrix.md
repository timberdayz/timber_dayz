# Model Comparison Matrix

| 维度 | 精品 | 精铺 | 铺货 |
|---|---|---|---|
| SKU 策略 | 少量核心 SKU | 中量筛选型 SKU 池 | 大量铺开 |
| 利润来源 | 单品利润、复购、品牌溢价 | 结构性效率和筛选 | 流量和规模 |
| 内容要求 | 高 | 中 | 低到中 |
| 商品页要求 | 高 | 中到高 | 中 |
| 供应链要求 | 高 | 中 | 中 |
| 价格战抗性 | 高于其他模式 | 中 | 低 |
| 适合 Shopee | 适合，但需搜索和评价承接 | 很适合 | 也适合 |
| 适合 TikTok | 很适合内容型精品 | 可做 | 弱于精品 |
| 组织复杂度 | 高 | 中 | 中 |
| 启动速度 | 慢 | 中 | 快 |
| 资本要求 | 中到高 | 中 | 中 |
| 品牌化可能性 | 高 | 中 | 低 |

## Core Trade-off

- 精品：慢，但深
- 精铺：平衡
- 铺货：快，但浅
