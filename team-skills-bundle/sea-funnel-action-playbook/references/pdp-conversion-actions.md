# PDP Conversion Actions

用于商品详情页和 listing 转化诊断。

## Matrix

| 数据表现 | 瓶颈判断 | 标准动作 |
|---|---|---|
| 主图点击低 | 第一眼价值不足 | 重拍主图，突出使用场景、结果、价格锚点 |
| 标题有流量但不准 | 关键词匹配弱 | 调整核心关键词、长尾词、国家语言表达 |
| 价格点击低 | 价格锚点不清 | 增加对比价、组合价、优惠显示 |
| 浏览高、咨询高、下单低 | 用户疑虑多 | 补 FAQ、规格说明、售后保障 |
| 浏览高、评价弱 | 信任不足 | 补买家秀、真实评价、评分运营 |
| 尺码 / 规格咨询多 | 信息不清 | 重做尺码表、规格图、场景说明 |
| 功能误解导致退货 | 预期管理失败 | 改标题、详情、视频，明确适用边界 |
| SKU 多但转化低 | 选择困难 | 减少无效 SKU，突出推荐款和组合 |

## Review Metrics

- CTR
- PDP conversion rate
- add-to-cart rate
- chat / consultation rate
- review count
- rating
- return reason
