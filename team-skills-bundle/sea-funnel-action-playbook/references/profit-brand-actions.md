# Profit And Brand Actions

用于判断产品是精品、流量品还是应淘汰。

## Matrix

| 数据表现 | 判断 | 标准动作 |
|---|---|---|
| 转化好、毛利差 | 流量品，不是精品 | 改供应链、提客单、做套装 |
| GMV 高、利润低 | 订单结构或成本问题 | 拆货本、物流、佣金、优惠、平台费 |
| 客单低、销量高 | 低价成交强 | 做组合、升级款、加购件 |
| 广告一停就没单 | 依赖付费流量 | 强化自然搜索、内容资产、评价 |
| 复购弱、评价弱 | 品牌资产不足 | 做包装、售后、内容、用户心智 |
| 差评集中在同一痛点 | 产品可改造或需止损 | 优先改产品，不要放量 |
| 价格带被打穿 | 红海风险 | 转差异化、转套装，或淘汰 |
| 有明确痛点且可改造 | 精品机会 | 做包装、规格、套装、人群定位 |

## Brand Decision

可进入精品验证的产品通常满足：

- 贡献利润可支撑内容和投放
- 有明确差异化路径
- 用户痛点可通过产品或表达改善
- 不依赖单次活动低价成交

不能作为精品的产品通常表现为：

- 只能靠低价成交
- 供应链没有改善空间
- 评价痛点无法修复
- 售后风险高
