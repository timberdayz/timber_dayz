# Shopee Ads Diagnostics

| 数据表现 | 判断 | 标准动作 |
|---|---|---|
| 曝光高、点击低 | 关键词或素材吸引力不足 | 改关键词、主图、价格锚点 |
| 点击高、转化低 | 商品页承接弱 | 优化 PDP、评价、保障、FAQ |
| 转化高、ROAS 低 | CPC 或利润结构问题 | 调出价、缩词、看利润结构 |
| ROAS 高、单量小 | 效率好但量级不足 | 试探性扩量，观察成本和利润 |
| 广告带单强、自然单弱 | 依赖广告 | 做 SEO、评价、自然流量承接 |
| 关键词烧钱不出单 | 词不准或页面不匹配 | 停词、换词、改标题和首图 |

## Not Recommended

- ROAS 高时不要直接大幅加预算，先看量级和利润。
- 曝光高点击低时不要先怪广告系统，先看素材和关键词。
