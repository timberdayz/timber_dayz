# TikTok Content Diagnostics

| 数据表现 | 判断 | 标准动作 |
|---|---|---|
| 播放高、3秒观看低 | 钩子弱 | 改开头 3 秒 |
| 3秒观看高、完播低 | 节奏弱 | 缩脚本、提早给结果 |
| 互动高、商品点击低 | 内容吸引人，但购买理由弱 | 改脚本，强化 CTA 和利益点 |
| 商品点击高、成交低 | 商品页或价格承接弱 | 优化 PDP、价格、评价、券 |
| 单条内容成交强 | 内容模型成立 | 复投、复制脚本、做达人扩散 |
| 内容强、利润差 | 流量型内容，不是精品内容 | 优化客单、套装、供应链 |

## Core Rule

内容数据必须拆成：

- 看得见
- 感兴趣
- 点得动
- 买得下

只要其中一环断，不能简单说“内容不行”。
