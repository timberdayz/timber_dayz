# Daily Checklist

## Start of Day

- [ ] 检查 GMV / 订单数 / 流量是否异常
- [ ] 检查利润是否异常
- [ ] 检查缺货和积压
- [ ] 检查退货和售后异常
- [ ] 判断今天是否存在优先级 1 问题
- [ ] 确定今天唯一主瓶颈

## During Day

- [ ] 至少完成 1 个核心增长动作
- [ ] 至少完成 1 个商品或内容优化动作
- [ ] 跟进库存 / 供应链 / 售后中的关键问题
- [ ] 如有选品任务，按 `sea-product-selection` 输出评分卡

## End of Day

- [ ] 记录今天的动作
- [ ] 记录今天的结果
- [ ] 标记未完成问题
- [ ] 确认明天优先级
