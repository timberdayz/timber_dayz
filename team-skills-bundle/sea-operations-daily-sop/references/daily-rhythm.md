# Daily Rhythm

标准精品运营日常节奏分为 6 段：

1. 早盘检查
2. 异常处理
3. 核心增长动作
4. 商品 / 内容优化
5. 库存 / 供应链 / 售后联动
6. 收尾复盘

## 1. 早盘检查

目标：快速确认今天是不是“正常工作日”还是“异常工作日”。

看：

- GMV / 订单数 / 流量是否异常
- 利润是否异常
- 爆款是否缺货
- 退货是否异常
- 广告 / 内容是否异常波动

## 2. 异常处理

如果存在优先级 1 异常，先处理异常，再谈增长动作。

## 3. 核心增长动作

根据主瓶颈，把今天最好的精力放在：

- 流量
- 转化
- 客单
- 利润
- 内容
- 选品

中的一个主方向。

## 4. 商品 / 内容优化

包括：

- 主图
- 标题
- 详情
- 评价
- FAQ
- 短视频
- 直播脚本
- 达人素材

## 5. 库存 / 供应链 / 售后联动

包括：

- 缺货
- 积压
- 退货
- 供应商反馈
- 包装 / 规格修订

## 6. 收尾复盘

当天结束前必须记录：

- 今天做了什么
- 哪个动作有效
- 哪个动作无效
- 明天优先级是什么
