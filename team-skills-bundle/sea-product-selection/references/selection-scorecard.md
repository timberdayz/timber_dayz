# Selection Scorecard

总分 100 分。

- 需求强度：25 分
- 竞争强度：20 分
- 利润空间：20 分
- 内容表现力：15 分
- 差异化 / 品牌化空间：10 分
- 供应链与风险：10 分

## Decision Rule

- `80-100`：`go`
- `65-79`：`hold`
- `<65`：`reject`

## 1. 需求强度（25）

看：

- 搜索热度
- 销量证据
- 30 / 60 天趋势
- 是否长期需求
- 是否存在明确痛点

评分建议：

- `20-25`：需求明确、稳定、有趋势支撑
- `12-19`：有需求，但仍需验证
- `0-11`：证据不足或需求弱

## 2. 竞争强度（20）

看：

- 同类卖家数量
- 头部评价壁垒
- 价格带压缩程度
- 内容竞争密度
- 是否存在可切入空档

评分建议：

- `15-20`：可切入，有空档
- `8-14`：竞争中等，需要更强差异化
- `0-7`：极度内卷，难切入

## 3. 利润空间（20）

至少估算：

- 预计售价
- 采购成本
- 物流成本
- 平台费用
- 优惠 / 推广 / 佣金
- 退款损失预估
- 预计贡献利润

评分建议：

- `15-20`：足以支撑试错、投流、活动
- `8-14`：利润一般，只适合轻验证
- `0-7`：利润过薄，不适合精品

## 4. 内容表现力（15）

看：

- 是否适合短视频
- 是否适合直播演示
- 是否适合达人带货
- 卖点是否能在 3 秒内讲清
- 是否存在明显场景或前后对比

评分建议：

- `12-15`：强内容型
- `7-11`：可做，但不强
- `0-6`：内容承接弱

## 5. 差异化 / 品牌化空间（10）

看：

- 是否能改包装
- 是否能改规格 / 套装
- 是否能做人群定位
- 是否能形成品牌表达

评分建议：

- `8-10`：明显可品牌化
- `4-7`：有一定空间
- `0-3`：基本只能卖同款

## 6. 供应链与风险（10）

看：

- MOQ
- 交期
- 质量稳定性
- 合规风险
- 售后 / 退货风险

评分建议：

- `8-10`：可控
- `4-7`：可做，但需风控
- `0-3`：风险过高
